# 🍪 Automating Cookie Clicker - Bot Development

A high-performance web automation bot designed to interact with the "Cookie Clicker" game using advanced scraping techniques. This project demonstrates proficiency in bypassing anti-bot mechanisms and handling dynamic web elements.

## 🚀 Key Features
* **Anti-Bot Bypass:** Utilizes `undetected-chromedriver` to mimic human behavior and avoid detection.
* **Dynamic Interaction:** Expertly handles JavaScript-rendered elements and pop-ups.
* **Robust Automation:** Implemented efficient wait-logic to manage asynchronous game events.

## 🛠️ Tech Stack
* **Language:** Python
* **Automation:** Selenium, Undetected-Chromedriver (UC)
* **Environment:** PyCharm (.venv), Jupyter Notebook
